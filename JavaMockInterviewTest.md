# Java Mock Interview Test

This file simulates an interview test for core Java preparation.

## Section 1: Theory (10 questions)
1. What is Java and why is it platform-independent?
2. What is the difference between JDK, JRE, and JVM?
3. What are primitive data types in Java?
4. What is the difference between `==` and `.equals()`?
5. What is encapsulation?
6. What is inheritance?
7. What is the difference between method overloading and overriding?
8. What is the difference between checked and unchecked exceptions?
9. What is the difference between `ArrayList` and `LinkedList`?
10. What is the difference between `HashMap` and `Hashtable`?

## Section 2: Coding Questions (10 questions)
1. Write a program to print even numbers from 1 to 20.
2. Write a program to reverse a string.
3. Write a program to find the factorial of a number.
4. Write a program to check whether a number is prime.
5. Write a program to find the largest number in an array.
6. Write a program to remove duplicates from an array.
7. Write a program to count vowels in a string.
8. Write a program to print the Fibonacci series.
9. Write a program to find the second largest number in an array.
10. Write a program to print the first 10 numbers of the Fibonacci series using recursion.

## Section 3: Scenario-Based Questions
1. How would you handle a null value in a Java application?
2. How would you make a class thread-safe?
3. How would you improve performance when using large collections?
4. How would you debug a NullPointerException?

## Mock Interview Tips
- Answer in short, structured points.
- Use one real-world example where possible.
- Practice coding without looking at notes.
