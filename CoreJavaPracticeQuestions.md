# Core Java Practice Questions

This file contains topic-wise practice questions for self-study and interview preparation.

## 1. Basics
1. What is Java?
2. What is the difference between JDK, JRE, and JVM?
3. Why is Java called platform-independent?
4. What is the purpose of the main method?

## 2. Variables and Data Types
1. What are primitive data types in Java?
2. What is the difference between int and double?
3. What is a boolean used for?
4. Write a program to declare and print variables of different types.

## 3. Operators
1. What is the difference between `==` and `=`?
2. What is the difference between `&&` and `||`?
3. Write a program using arithmetic operators.
4. Write a program using comparison and logical operators.

## 4. Control Statements
1. What is the difference between if-else and switch?
2. Write a program to check whether a number is positive, negative, or zero.
3. Write a program to print the day of the week using switch.

## 5. Loops
1. What is the difference between for, while, and do-while loops?
2. Write a program to print numbers from 1 to 10 using a for loop.
3. Write a program to print the factorial of a number.

## 6. Arrays and Strings
1. What is an array?
2. What is the difference between an array and an ArrayList?
3. Write a program to find the largest element in an array.
4. What is the difference between `String` and `StringBuilder`?
5. Write a program to reverse a string.

## 7. Methods
1. What is a method?
2. What is method overloading?
3. Write a method to add two numbers.
4. Write a method to find the sum of digits of a number.

## 8. OOP Concepts
1. What is a class?
2. What is an object?
3. What is encapsulation?
4. What is inheritance?
5. What is polymorphism?
6. What is abstraction?
7. Write a class with a constructor and object creation example.

## 9. Exception Handling
1. What is an exception?
2. What is the difference between throw and throws?
3. Write a program to handle division by zero.
4. What is the purpose of the finally block?

## 10. Collections
1. What is the difference between List, Set, and Map?
2. What is the difference between ArrayList and LinkedList?
3. What is the difference between HashMap and Hashtable?
4. Write a program to add and print elements in an ArrayList.
5. Write a program to store key-value pairs in a HashMap.

## 11. Multithreading
1. What is a thread?
2. What is the difference between `start()` and `run()`?
3. Write a simple thread example.
4. What is synchronization?

## 12. Java 8+
1. What is a lambda expression?
2. What is a functional interface?
3. What is the Stream API?
4. Write a program to filter even numbers using streams.

## 13. Interview Practice
1. Explain Java in one minute.
2. Explain OOP concepts in simple words.
3. Describe the difference between `equals()` and `==`.
4. Explain how collections are used in real-world applications.
