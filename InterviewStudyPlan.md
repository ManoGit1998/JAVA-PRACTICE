# 2-Week Java Interview Study Plan

## Week 1: Core Java Basics

### Day 1
- Learn Java basics, JDK, JRE, JVM
- Write a simple Hello World program

### Day 2
- Practice variables, data types, and operators
- Understand type conversion

### Day 3
- Learn conditional statements and loops
- Solve 10 small coding questions

### Day 4
- Study arrays and strings
- Practice string manipulation programs

### Day 5
- Learn methods and method overloading
- Practice function-based problems

### Day 6
- Study classes, objects, constructors
- Practice object-oriented programming examples

### Day 7
- Revise all topics from Week 1
- Write short answers for common interview questions

## Week 2: Advanced Core Java

### Day 8
- Study inheritance, polymorphism, abstraction, and encapsulation
- Compare interface vs abstract class

### Day 9
- Learn exception handling
- Practice try-catch-finally examples

### Day 10
- Study collections: List, Set, Map
- Practice code using ArrayList, HashMap, HashSet

### Day 11
- Learn multithreading basics
- Understand thread lifecycle

### Day 12
- Study file handling and input/output
- Practice basic file read/write operations

### Day 13
- Revise important interview questions
- Practice 20 short answers aloud

### Day 14
- Mock interview day
- Solve coding problems under time pressure

## Daily Routine
- Study for 2 hours
- Write 1-2 code examples
- Revise previous notes
- Practice speaking answers out loud
