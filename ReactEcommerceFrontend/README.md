# React Ecommerce Frontend Starter

This folder contains a simple React + Vite starter app for an e-commerce frontend.

## Structure
- auth: login and authentication UI
- catalog: product listing and detail pages
- cart: cart experience
- checkout: checkout flow
- orders: order history and tracking UI

## Run locally
1. Open this folder in terminal
2. Run:
   ```bash
   npm install
   npm run dev
   ```
3. Open the local URL shown by Vite
