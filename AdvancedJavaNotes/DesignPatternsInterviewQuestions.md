# Design Patterns Interview Questions and Answers

## 1. What is a design pattern?
A design pattern is a reusable solution to a common software design problem.

## 2. Why are design patterns used?
They make code more maintainable, scalable, and easier to understand.

## 3. What is the Singleton pattern?
Singleton ensures that only one instance of a class is created and used throughout the application.

## 4. When do you use the Singleton pattern?
Use it for logging, configuration, or shared resources.

## 5. What is the Factory pattern?
Factory pattern creates objects without exposing the creation logic to the client.

## 6. What is the difference between Factory and Abstract Factory?
Factory creates one type of object, while Abstract Factory creates families of related objects.

## 7. What is the Builder pattern?
Builder is used to construct complex objects step by step.

## 8. When do you use the Builder pattern?
Use it when an object has many optional fields or complex initialization.

## 9. What is the Adapter pattern?
Adapter makes incompatible interfaces work together.

## 10. What is the Decorator pattern?
Decorator adds behavior to an object dynamically without changing its structure.

## 11. What is the Facade pattern?
Facade provides a simple interface to a complex subsystem.

## 12. What is the Proxy pattern?
Proxy controls access to another object and is often used for lazy loading and security.

## 13. What is the Strategy pattern?
Strategy defines interchangeable algorithms or behaviors.

## 14. What is the Observer pattern?
Observer allows objects to be notified when another object changes state.

## 15. What is the Command pattern?
Command encapsulates a request as an object, enabling parameterization and undo/redo support.

## 16. What is the Template Method pattern?
Template Method defines the skeleton of an algorithm and lets subclasses override certain steps.

## 17. Which design patterns are most commonly used in Java?
Singleton, Factory, Builder, Adapter, Decorator, Strategy, and Observer are very common.

## 18. Which design pattern is used in Spring Boot?
Spring Boot heavily uses Dependency Injection and Factory-like bean creation patterns.

## 19. What is the difference between Adapter and Decorator?
Adapter changes the interface of an existing class, while Decorator adds behavior without changing the interface.

## 20. What is the difference between Singleton and Static class?
Singleton is an object-based pattern with an instance and lifecycle, while static class is just utility-like and cannot be instantiated.
