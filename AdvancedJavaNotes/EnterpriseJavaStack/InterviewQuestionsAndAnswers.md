# Interview Questions and Answers

## Spring
### Q1. What is Dependency Injection?
A. Dependency Injection is a design pattern where dependencies are supplied from outside the class.

### Q2. What is the difference between Spring and Spring Boot?
A. Spring is a framework, while Spring Boot is an extension that simplifies configuration and setup.

## Spring Boot
### Q1. What is Spring Boot?
A. Spring Boot is a framework that helps create production-ready Spring applications quickly.

### Q2. What is auto-configuration?
A. Auto-configuration automatically configures beans based on dependencies present in the classpath.

## Microservices
### Q1. What is a microservice?
A. A microservice is a small, independently deployable service that performs a specific business capability.

### Q2. What is an API Gateway?
A. An API Gateway routes requests to appropriate services and handles cross-cutting concerns.

## Docker
### Q1. What is Docker?
A. Docker is a platform for building, shipping, and running applications inside containers.

### Q2. What is the difference between a container and a VM?
A. Containers share the host OS kernel, while VMs have their own guest OS.

## Kubernetes
### Q1. What is Kubernetes?
A. Kubernetes is a container orchestration platform used to deploy and manage containerized applications.

### Q2. What is a Pod?
A. A Pod is the smallest deployable unit in Kubernetes.

## Kafka
### Q1. What is Kafka?
A. Kafka is a distributed event streaming platform used for real-time messaging and data pipelines.

### Q2. What is a partition?
A. A partition is a unit of parallelism in Kafka that stores a subset of messages for a topic.
