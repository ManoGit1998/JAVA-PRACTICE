# Advanced Java Interview Questions

## Core Questions
1. What is the difference between HashMap and ConcurrentHashMap?
2. What is the difference between synchronized and volatile?
3. What is a deadlock?
4. What is the difference between Comparable and Comparator?
5. What is the difference between `==` and `.equals()`?
6. What is the purpose of the Stream API?
7. What is the difference between `filter` and `map` in streams?
8. What is garbage collection?
9. What is a functional interface?
10. What is Dependency Injection?

## Short Answers
- A deadlock happens when two or more threads wait forever for each other.
- `Comparable` provides natural ordering; `Comparator` provides custom ordering.
- Streams process collections functionally and concisely.
- Dependency Injection means supplying dependencies from outside the class.
