# Java Daily Coding Practice Sheet

Use this sheet every day for 30-45 minutes.

## Day 1
1. Write a program to print "Hello Java".
2. Write a program to add two numbers.
3. Write a program to find the square of a number.

## Day 2
1. Write a program to check if a number is even or odd.
2. Write a program to print numbers from 1 to 10 using a loop.
3. Write a program to find the sum of first 10 natural numbers.

## Day 3
1. Write a program to reverse a string.
2. Write a program to check if a string is palindrome.
3. Write a program to count vowels in a string.

## Day 4
1. Write a program to find the largest element in an array.
2. Write a program to find the second largest element.
3. Write a program to sort an array.

## Day 5
1. Write a program to create a class and object.
2. Write a program demonstrating constructor overloading.
3. Write a program demonstrating inheritance.

## Day 6
1. Write a program to handle division by zero with exception handling.
2. Write a program using try-catch-finally.
3. Write a program to create a custom exception.

## Day 7
1. Write a program using ArrayList.
2. Write a program using HashMap.
3. Write a program using HashSet.

## Day 8
1. Write a program using a lambda expression.
2. Write a program using streams to filter even numbers.
3. Write a program using Optional.

## Daily Practice Rules
- Write code without copying.
- Try to solve each problem in 10-15 minutes.
- Review your mistakes after every session.
