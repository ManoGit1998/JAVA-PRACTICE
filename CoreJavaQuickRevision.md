# Core Java Quick Revision

## 1. Java Basics
- Java is object-oriented and platform-independent.
- JVM executes bytecode.
- JDK is for development; JRE is for running applications.

## 2. Variables
- int, double, char, boolean are common primitive types.
- Variables must have a type.

## 3. Control Flow
- `if`, `else`, `switch` are decision-making statements.
- `for`, `while`, `do-while` are loops.

## 4. OOP
- Class = blueprint
- Object = instance
- Encapsulation = data hiding
- Inheritance = reuse of code
- Polymorphism = one interface, many forms
- Abstraction = hiding implementation details

## 5. Methods
- Methods are reusable code blocks.
- Overloading means same name, different parameters.

## 6. Exceptions
- Use try-catch-finally.
- throw is used to raise an exception.
- throws declares exceptions.

## 7. Collections
- List: ordered, duplicates allowed
- Set: unique elements
- Map: key-value pairs

## 8. Multithreading
- Thread is a unit of execution.
- Use synchronization to avoid race conditions.

## 9. Java 8+
- Lambda expressions simplify code.
- Streams process collections functionally.
- Optional avoids null issues.
