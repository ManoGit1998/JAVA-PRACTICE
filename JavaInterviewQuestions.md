# Java Interview Questions with Answers

## Basic Questions

### 1. What is Java?
Java is a high-level, object-oriented, platform-independent programming language.

### 2. What is the difference between JDK, JRE, and JVM?
- JDK: development tools
- JRE: runtime environment
- JVM: executes Java bytecode

### 3. What is the main method?
The main method is the entry point of a Java program.

### 4. What is a class?
A class is a blueprint for creating objects.

### 5. What is an object?
An object is an instance of a class.

## OOP Questions

### 6. What is encapsulation?
Encapsulation means hiding data and exposing access through methods.

### 7. What is inheritance?
Inheritance allows one class to acquire the properties and methods of another class.

### 8. What is polymorphism?
Polymorphism means one interface can behave differently in different classes.

### 9. What is abstraction?
Abstraction hides implementation details and shows only necessary features.

## Java Core Questions

### 10. What is the difference between `==` and `.equals()`?
`==` compares references for objects and values for primitives. `equals()` compares content.

### 11. What is the difference between `String` and `StringBuilder`?
`String` is immutable; `StringBuilder` is mutable.

### 12. What is method overloading?
Same method name with different parameters.

### 13. What is method overriding?
A subclass provides a specific implementation of a method already defined in the parent class.

### 14. What is the difference between abstract class and interface?
An abstract class can have both abstract and concrete methods, while an interface mainly defines behavior.

## Collections Questions

### 15. What is the difference between `ArrayList` and `LinkedList`?
`ArrayList` is faster for indexing; `LinkedList` is better for insertion and deletion.

### 16. What is the difference between `HashMap` and `Hashtable`?
`HashMap` is not synchronized and allows null values, while `Hashtable` is synchronized and does not allow null.

### 17. What is the difference between `List`, `Set`, and `Map`?
- List: ordered and allows duplicates
- Set: unique elements
- Map: key-value pairs

## Exception Handling

### 18. What is an exception?
An exception is an event that disrupts the normal flow of a program.

### 19. What is the purpose of `finally`?
The `finally` block always executes whether an exception occurs or not.

### 20. What is the difference between checked and unchecked exceptions?
Checked exceptions are checked at compile time; unchecked are runtime exceptions.

## Multithreading

### 21. What is a thread?
A thread is the smallest unit of execution in a program.

### 22. What is the difference between `start()` and `run()`?
`start()` creates a new thread; `run()` executes the task in the current thread.

## Final Practice Tips

- Write code manually from memory.
- Explain each concept in your own words.
- Practice one question and one coding example daily.
